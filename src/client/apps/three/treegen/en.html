Parameter Descriptions:


  • Max Slices:
    controls the maximum number of branch sub-sections that are allowed to be rendered

  • Split Chance:
    controls the probability of a new branch splitting off of a already existing branch

  • Angle Mod:
    modifies how much a branch can change its angle by for each sub-section

  • Height Mod:
    multiplies the height of the branch-subsections, making the tree taller

  • Width Mod:
    multiplies the speed at which branches spread out, making the tree wider

  • Starting Size:
    Changes the radius of the stump of the tree, making the tree wider or slimer around the base

  • Size Decrement:
    Sets the amount by which the branches get reduced in width for each sub-section

  • Branch Variance:
    Changes the percentage in width that a child branch can be smaller than a parent branch by. Generally makes trees look more "natural" and random.

  • Sections:
    controls the number of faces that each sub-section of a branch has. Increase to produce more rounded looking branches
